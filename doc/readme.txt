Серверное приложение, позволяющее транслировать данные с внешнего аналого-цифрового преобразователя клиентам локально-вычислительной сети. 
Основным функциональным элементом системы сбора данных является АЦП Е-140 фирмы L-Card. Данное устройство является малогабаритным многофункциональным измерительным модулем, подсоединяемым к ПК через USB-интерфейс. 
Базовые функции Е-140: 
- многофункциональное АЦП с мультиплексированием каналов;
- цифровой асинхронный ввод/вывод.
Многоканальное 14-ти разрядное АЦП модуля Е-140 позволяет работать с 16-ю дифференциальными или 32-мя каналами с общей землей. 
Модуль обеспечивает непрерывный сбор аналоговых данных на частотах дискретизации АЦП до 100.0кГц. Наличие специального входа/выхода синхронизации позволяет соединить несколько Е-140 по этим линиям по схеме «один ведущий – несколько ведомых», 
позволяя аппаратно засинхронизировать моменты старта АЦП в нескольких модулях Е-140.
	
Аналоговая первичная информация поступает от источника на АЦП, которая затем по средствам шины USB передается на ПК. 
Далее рассматривая программную часть можно выделить три основных этапа обработки информации. На первом этапе происходит прием и обработка данных с целью выделения первичной информации в цифровом виде. 
Далее на втором этапе происходит непосредственно обработка первичной информации для приведения ее к заданному формату. И в конце следует этап передачи данных по сети на клиентские машины.
